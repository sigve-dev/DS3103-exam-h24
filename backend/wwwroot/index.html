Welcome to Trump API

Without further ado I attempted to create my Web API Doc based upon the Rick and Morty API. Drawing inspiration from that. With this API there will be 8 total items stored in the API available to use. With the ability to create more, or delete all of them. Please, enjoy my attempt at the DS3103 Exam H24. Merry Christmas!

REST

"Merch": "http://localhost:5255/api/TrumpMerch",

"Staff": "http://localhost:5255/api/TrumpStaff/all"

Merch

    Merch Schema

  • Id: int | Id of the merch
  • Type: string | Type of merch (clothing/accessory)
  • Image: string | Image of the merch
  • Quantity: int | How many are in stock
  • Price: int | The price of the merch
  • Details: string | Details about the product

Endepunkter

Get all Merch: "http://localhost:5255/api/TrumpMerch";

Get Merch by Id: "`http://localhost:5255/api/TrumpMerch/{id}`"

Get Merch by name: "`http://localhost:5255/api/TrumpMerch/by-name/{name}`"

Staff

    Staff Schema

  • Id: int | Id of the staff
  • Name: string | Staffmembers name
  • Department: string | Department the staff belongs
  • Position: string | Which position

Endepunkter

Get all staff: "http://localhost:5255/api/TrumpStaff/all"
